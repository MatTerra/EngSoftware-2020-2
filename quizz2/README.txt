O arquivo engsoft.txt disponibilizado no aprender3 está em um encoding diferente do UTF-8, que é o padrão do ruby.
Para simplificar a solução, o arquivo foi convertido para UTF-8 e disponibilizado como parte deste arquivo. 

Para executar os testes, execute
ruby tests/*.rb

Para verificar a saída do processamento do arquivo, execute
ruby src/main.rb
